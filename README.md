# Sifanlian Desktop

这是 sifanlian.me 的桌面客户端应用程序。

## 开发说明

### 环境要求
- Node.js (推荐 v16 或更高版本)
- npm 或 yarn

### 安装依赖
```bash
npm install
```

### 运行开发版本
```bash
npm start
```

### 构建应用
```bash
npm run build
```

构建完成后，可以在 `dist` 目录下找到打包好的应用程序。

## 功能特性
- 将 sifanlian.me 网站包装成桌面应用
- 无需浏览器即可访问
- Windows 系统一键安装

## 注意事项
- 应用需要网络连接才能正常运行
- 首次启动可能需要稍等片刻加载
